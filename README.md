# Blinking 67

A monumental C++23 achievement: a blinking `67` rendered in oversized ASCII glory, changing color and style like the final, triumphant flourish of Western civilization.

## Overview

This program clears the terminal, prints a large ASCII-art `67`, makes it blink, and rotates through different pseudo-fonts and ANSI colors. In strictly practical terms, it does almost nothing. In cultural terms, it does everything.

## Build

Compile with a C++23-compatible compiler.

### macOS

For the citizens of macOS, this treasure may be built in Terminal with either `clang++` or `g++`, depending on what is installed on the machine.

Using `clang++`:

```bash
clang++ -std=c++23 -O2 -o blinking_67 main.cpp
```

Using `g++`:

```bash
g++ -std=c++23 -O2 -o blinking_67 main.cpp
```

If no compiler is available, install Apple's Command Line Tools first:

```bash
xcode-select --install
```

### Linux

For Linux users, the path to enlightenment is equally direct.

```bash
g++ -std=c++23 -O2 -o blinking_67 main.cpp
```

If `g++` is missing, install it with the package manager of the distribution and proceed without fear.

### Windows

```bash
g++ -std=c++23 -O2 -o blinking_67.exe main.cpp
```

## Run

### macOS

```bash
./blinking_67
```

### Linux

```bash
./blinking_67
```

### Windows

```bash
blinking_67.exe
```

## One-Line Invocation for the Impatient

For those who believe civilization should move faster, Linux and macOS can compile and run the program in a single command:

```bash
g++ -std=c++23 -O2 -o blinking_67 main.cpp && ./blinking_67
```

## Technical Notes

The program uses:
- `std::this_thread::sleep_for` for the blink timing
- ANSI escape sequences for colors and screen clearing
- Multiple ASCII-art variants to simulate font changes
- Optional Windows console setup to enable ANSI support
